# Data Knowledge Repository

`yangjin2021/data` 是数据、资料、知识库和个人研究材料的主仓库。当前 `main` 分支已经重新整理为一个主题化知识库，根目录只保留入口说明和来源映射，实际资料统一放在 `knowledge/` 下。

## Structure

| Path | Content |
|---|---|
| `knowledge/business-theory/` | 商业模型、商业分析模板、模型选择速查表 |
| `knowledge/fba-and-analysis/` | 亚马逊 FBA/FBM 选品评分体系、纵横分析法 |
| `knowledge/cybernetics/` | 控制论、信息论、系统科学、最优控制资料库 |
| `knowledge/life-50-books/` | 50 本重要书籍的合法资源索引与书目数据 |
| `knowledge/flomo-notes/` | flomo 外部链接与导入资料 |
| `knowledge/trading-and-python-data/` | 交易策略笔记、Python 环境文件、相关资源 |

## Rules

- 新资料优先放入最接近的 `knowledge/<topic>/` 目录。
- 不再把历史分支整份放入 `branch-archives/`；有价值内容已经提升到正式主题目录。
- FBA 更新脚本继续使用 `knowledge/fba-and-analysis/FBA评分体系.txt` 作为远端目标文件。
- 项目代码、Codex skills 和临时缓存不要放入本仓库。

## Map

历史来源和整理规则记录在 `SOURCE_MAP.md`。
