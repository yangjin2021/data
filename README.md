# Data Knowledge Repository

这个仓库是 yangjin2021 账号的数据与知识资料主仓库。`main` 分支已经整合了原先分散在 data、data-private、think-model-public 等仓库和分支里的资料内容。

## 内容结构

- `knowledge/fba-and-analysis/`：FBA 评分体系与横纵分析法资料。
- `knowledge/cybernetics-pdfs/`：控制论、信息论、系统科学相关 PDF 资料。
- `knowledge/cybernetics-ocr-vault/`：控制论资料的 Obsidian/OCR 知识库版本。
- `knowledge/trading-and-python-data/`：交易策略、Python 环境资料和历史数据整理。
- `docs/source-metadata/`：迁移过程保留的分支说明文件。
- `branch-archives/`：删除远端分支前保存的各分支完整快照。

## 使用建议

把新增数据、文档、PDF、研究材料、知识库都放到这个仓库的 `main` 分支。项目代码和 Codex skills 不再放这里。

非 `main` 分支已经统一归档到 `branch-archives/`，后续不再依赖远端分支保存数据。
