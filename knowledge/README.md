# Knowledge Index

这个目录是仓库的正式资料区。每个一级目录代表一个主题或资料类型。

| Directory | Use for |
|---|---|
| `business-theory/` | 商业模型、商业分析方法、报告模板 |
| `fba-and-analysis/` | 亚马逊选品评分、FBA/FBM 判断、纵横分析法 |
| `cybernetics/` | 控制论、系统科学、信息论、最优控制资料 |
| `life-50-books/` | 50 本重要书籍的合法资源索引 |
| `flomo-notes/` | flomo 外部链接整理 |
| `trading-and-python-data/` | 交易策略、Python 环境、相关依赖资源 |

新增资料时，优先放入已有主题目录；只有确实无法归类时再新建一级主题目录。
