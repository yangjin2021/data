# 一、规则定位

本规则用于判断亚马逊商品是否适合继续做 FBA 或 FBM 选品。

v7.4 的核心变化：

1. 保留 v7.3：不再使用“维度1/维度2”叫法。
2. 评分表只显示：产品属性与趋势、预估利润与投产比。
3. 保留表格优先输出：评分表第一，1688货源分析表第二，否决项与推荐指数第三。
4. 新增“品牌识别规则”。
5. 标题翻译后仍保留、且不像规格/型号/通用词的英文，优先识别为品牌或疑似品牌。
6. Compatible / Fit / Fits / Suitable for / Replacement for / Intended for 后面的品牌，只作为兼容对象，不自动判为侵权。
7. 但兼容对象必须真实、可验证，且不得出现 Logo、原厂包装、官方授权暗示或仿冒外观。

核心原则：

> 先识别品牌，再判断是否侵权。  
> 有品牌不等于一定否决；但品牌不清、货源品牌不一致、1688带品牌Logo或原包装，必须进入高风险或否决处理。

# 二、内部分析顺序

虽然最终输出先给表格，但内部分析必须按以下顺序完成：

1. 读取 Amazon 产品信息。
2. 读取 1688 货源信息。
3. 识别品牌和疑似品牌。
4. 检查否决项。
5. 分析产品属性与趋势。
6. 分析预估利润与投产比。
7. 分析1688货源匹配度、低价陷阱、采购单位、品牌一致性和供应商风险。
8. 得出推荐指数与最终结论。
9. 最后按“输出顺序”展示。

# 七、品牌识别规则｜必须执行

## 7.1 品牌识别的核心方法

判断一个 Amazon 产品是否有品牌，必须优先看以下位置：

1. Amazon 页面 Brand 字段。
2. Amazon 标题开头或标题中明显独立的英文词。
3. Manufacturer / Part Number / Model / Included Components 字段。
4. 图片、包装、产品本体是否出现 Logo 或品牌字样。
5. 1688 货源标题、图片、包装、产品本体是否出现品牌字样。

## 7.2 标题翻译法

当 Amazon 标题翻译成中文后：

- 如果某个英文词翻译不出来，且不像规格、型号、通用品类词，一般优先识别为品牌名或疑似品牌名。
- 这类英文词常出现在标题开头，例如：`DMITIZYN NXC-18 AC Contactor...` 中的 `DMITIZYN`。
- 不能因为它不是知名大牌，就忽略品牌风险。

## 7.3 哪些英文不像品牌，通常是规格/型号/通用词

以下通常不直接按品牌识别，而应先按型号、规格、标准、品类词处理：

| 类型 | 示例 |
|---|---|
| 型号 | NXC-18、CJX2-1810、CJX2-1801、LC1D、B0FCMB3HFS |
| 规格 | 18A、24V、36V、110V、220V、380V、3-Phase、3P、1NO、1NC |
| 安装方式 | DIN Rail、Panel Mount、Unit Mount |
| 品类词 | AC Contactor、Motor Contactor、Relay、Switch、Film Sheet、Bracket |
| 材质/功能词 | Polyimide、Stainless Steel、Non-Adhesive、Heat Resistant |

## 7.4 哪些英文更像品牌或疑似品牌

以下情况优先识别为品牌或疑似品牌：

- 标题开头出现独立英文词，翻译后仍保留英文。
- 英文词不表示材质、尺寸、电压、电流、型号、接口、规格或通用产品名。
- Brand 字段与标题中的英文词一致或接近。
- Manufacturer 字段与该英文词一致或接近。
- 该词出现在产品图片、包装、说明书或产品本体上。

示例：

| 标题片段 | 判断 |
|---|---|
| DMITIZYN NXC-18 AC Contactor | DMITIZYN 为品牌或疑似品牌；NXC-18 为型号 |
| zQIhAoN Polyimide Film Sheet | zQIhAoN 为品牌或疑似品牌；Polyimide Film Sheet 为品类 |
| uxcell Brass Sheet Roll | uxcell 为品牌；Brass Sheet Roll 为品类 |
| CHINT NXC-18 AC Contactor | CHINT/正泰为品牌；NXC-18 为型号 |

# 八、品牌风险处理规则

## 8.1 有品牌不等于直接否决

识别出品牌后，不要立刻否决。必须进一步判断：

| 品牌状态 | 处理 |
|---|---|
| Amazon 自有品牌 / 小品牌 / 白牌品牌 | 不直接否决，但1688货源必须能做中性包装或同品牌授权货 |
| Amazon品牌与1688品牌不一致 | 高风险，推荐指数下降 |
| Amazon是DMITIZYN，1688是正泰/CHINT/上海人民等品牌货 | 不建议直接用，品牌不一致且易形成假冒/错配风险 |
| 1688货源带第三方Logo、品牌包装、原厂包装 | 直接排除该货源 |
| 产品本体和包装可以无Logo、中性包装 | 可继续验证 |
| 无法确认是否带Logo/原包装 | 否决项状态至少为待确认 |

## 8.2 以下情况属于明确否决或强排除

- 1688货源图片、包装、产品本体带第三方品牌Logo，且你没有授权。
- 1688货源为正泰/CHINT、上海人民、Schneider、Siemens、ABB、Omron 等品牌件，但 Amazon 目标产品不是该品牌或你没有授权。
- 商品标题、图片或包装暗示 Original / Genuine / Official / Authorized / OEM / 原装 / 正品 / 官方 / 授权，无法证明授权。
- 以某个品牌 ASIN 销售，但实际采购无品牌或其他品牌货。
- 产品外观、包装、颜色组合、图案或结构明显仿冒知名品牌。

## 8.3 品牌待确认时的推荐指数限制

| 情况 | 推荐指数上限 |
|---|---:|
| 标题存在疑似品牌，但1688货源可中性包装 | 75 |
| Amazon品牌与1688品牌不一致 | 65 |
| 1688货源疑似带品牌Logo但未确认 | 60 |
| 1688明确带第三方Logo或原包装，无授权 | 39 |
| 产品还涉及电气、医疗、儿童、安全等高责任风险 | 在上述基础上继续下调 |

# 九、兼容表达处理规则

Compatible For / Fit For / Fits / Replacement For / Suitable For / Intended For 后面的品牌或型号，不自动等同于品牌侵权。

## 9.1 不自动否决的情况

如果只是说明真实兼容关系，且满足以下条件，可以进入谨慎评估：

- 标题格式为：自有品牌/Generic + 产品名 + Compatible/Fit/Replacement for + 第三方品牌型号。
- Brand 字段使用自己的品牌或 Generic，不使用第三方品牌。
- 主图、详情图、包装、产品本体不出现第三方Logo。
- 不使用 Original / Genuine / Official / Authorized / 原装 / 正品 / 官方 / 授权 等表达。
- 1688供应商能提供中性包装、无Logo实拍图、规格和兼容型号确认。
- 兼容关系真实、可验证，不夸大适配范围。

## 9.2 必须否决或强降级的情况

- Compatible 后面的品牌被放进 Brand 字段。
- 主图、包装或产品本体出现第三方 Logo。
- 标题或图片暗示原装、正品、官方、授权，但无法提供授权。
- 兼容关系无法验证，适配范围过宽。
- 差评集中在不兼容、装不上、不识别、质量差。

# 十二、低价陷阱规则

以下价格不能直接作为真实采购成本：

- 0.1元、0.5元、1元、2元、3元。
- 询价价。
- 引流价。
- 起批量很高但价格极低。
- 复杂规格却统一低价。
- 同类产品普遍几十元，但某个商品只要几元。
- 标题和图片不一致。
- 商品规格没有明确写清楚。

处理方式：

- 不能用最低异常价计算利润。
- 异常低价只能作为“有货线索”。
- 默认选择较可信的中位价格或最接近同款的价格计算。
- 如果无法确认真实价格，“预估利润与投产比”单项达成率不得高于50%。
