# 版本历史

当前最新版本：`v6.1`  
当前执行方式：`v6.0` + `v6.1`

## 版本总览

| 版本 | 版本名 | 修改的地方 | 状态 |
|---|---|---|---|
| v1.0 | 基础规则 | 建立维度0否决项 + 维度1-4综合评估；统一使用权重、单项达成率、加权贡献、综合达成率。 | 已归档 |
| v1.1 | 1688序号 | 增加1688候选产品字段抽取；要求保留截图原始序号；截图序号只做溯源，不代表推荐排序。 | 已归档 |
| v2.0 | 校准机制 | 增加原始综合达成率和校准后综合达成率；最终结论参考校准后结果，但否决项、硬性红线、真实利润优先。 | 已归档 |
| v2.1 | 多表格输出 | 固定输出结构：百分比评估表、1688推荐、否决项评价、详细分析、最终结论。 | 已归档 |
| v3.0 | 品牌风险 | 强化品牌、商标、IP、授权风险；第三方Logo、原厂暗示、授权暗示进入重点检查。 | 已归档 |
| v3.1 | 兼容表达 | 明确 Compatible For / Fit For / Replacement For / Suitable For 等兼容表达不自动等同侵权；无Logo、无原厂包装、无授权暗示时可谨慎评估。 | 已归档 |
| v3.3 | 人读 + AI执行 | 将规则整理成 Markdown 说明 + YAML 规则块；提升可读性和执行稳定性。 | 已归档 |
| v4.1 | 兼容最终 | 权重合计调整为140%；禁用校准后综合达成率；强调原始综合达成率、真实利润和售后全损。 | 已归档 |
| v4.1.1 | 权重修正 | 修正权重表达；明确投产比权重为70%，产品属性21%，风险承受21%，1688匹配28%。 | 已归档 |
| v4.2 | 重量系统 | 增加实际重量、体积重、计费重量、跨境运费、运费占销售额比例；识别抛货、偏重、重货、低价重货风险。 | 已归档 |
| v5.0 | 成本闸门过渡 | 从重量系统过渡到截图初筛、成本闸门、利润压力测试；当前未找到独立完整原文，只保留占位。 | 已归档 |
| v6.0 | 截图初筛成本闸门 | 一张Amazon截图 + 一张1688截图即可初筛；新增保守默认值、采购单位换算、默认计费重量、利润闸门、三重压力测试、成本可信度。 | 基础版本 |
| v6.1 | 类目退货率 | 新增类目退货率预设系统；类目作为默认退货率和售后全损预留来源之一；低退货率类目适度提高投产比，高退货率类目降低投产比。 | 最新版本 |
