# Trading And Python Data

这里保存交易策略笔记、Python 环境文件和相关本地资源。

## Structure

| Directory | Content |
|---|---|
| `strategies/readable-filenames/` | 文件名正常可读的交易策略笔记 |
| `strategies/legacy-encoded-filenames/` | 旧导入中保留下来的乱码文件名策略笔记 |
| `python-environment/` | Python requirements、TA-Lib wheel、安装说明 |
| `resources/` | 字体、电脑依赖等历史资源 |
| `tools/` | 上传或同步相关脚本片段 |

旧导入文件名没有批量硬改，以免误伤引用；目录层级已整理为可读结构。
