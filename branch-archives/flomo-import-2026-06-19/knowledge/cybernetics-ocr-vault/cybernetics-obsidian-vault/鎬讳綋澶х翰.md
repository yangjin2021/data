# 总体大纲

## 1. 控制论基础层

- [[概念/控制论]]：系统、目标、调节、反馈、通信和控制的一般框架。
- [[概念/系统]]：整体、边界、输入输出、状态和环境。
- [[概念/反馈]] / [[概念/前馈]]：闭环校正与预先补偿。
- [[概念/稳定性]]：控制系统可用性的第一约束。

## 2. 信息与信号层

- [[概念/信息论]]、[[概念/熵]]、[[概念/信道容量]]：控制所需信息的度量和限制。
- [[概念/编码]]：效率与可靠性的技术桥梁。
- [[概念/信号处理]]、[[概念/采样定理]]、[[概念/滤波]]：观测信号进入控制系统前的处理链条。

## 3. 系统数学层

- [[概念/线性系统]]、[[概念/非线性系统]]：两类基础动态对象。
- [[概念/传递函数]]、[[概念/信号流图]]：输入输出和结构图表示。
- [[概念/状态空间]]：现代控制的统一语言。
- [[概念/可控性]]、[[概念/可观测性]]：系统能否被控制和被认识的结构条件。

## 4. 现代控制层

- [[概念/状态反馈]]：通过状态变量塑造闭环极点和动态品质。
- [[概念/观测器]]：状态不可全测时的模型化估计。
- [[概念/Lyapunov稳定性]]：稳定性证明和非线性系统分析工具。
- [[概念/Kalman滤波]]：随机噪声下的状态估计。

## 5. 最优与决策层

- [[概念/最优控制]]：动态约束下的性能优化。
- [[概念/极大值原理]]：候选最优轨迹的必要条件。
- [[概念/动态规划]] 与 [[概念/HJB方程]]：全局最优的递推/偏微分表达。
- [[概念/LQR]] 与 [[概念/Riccati方程]]：线性二次型最优调节。
- [[概念/最优停止]]：何时停止观察或行动的序贯决策理论。

## 6. 适应、鲁棒与设计层

- [[概念/自适应控制]] 与 [[概念/参数辨识]]：对象变化时的在线学习控制。
- [[概念/灵敏度]] 与 [[概念/鲁棒性]]：参数扰动、模型误差与设计裕度。
- [[概念/优选法]] 与 [[概念/最优设计]]：从试验搜索到工程优化。

## 7. 不确定性与质量治理层

- [[概念/模糊测度]]：处理非概率不确定性。
- [[概念/质量控制]] 与 [[概念/控制图]]：用统计方法监控过程变差。
