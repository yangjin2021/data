# 合法资源来源

本分支使用以下类型的合法入口：

1. Project Gutenberg：公版电子书入口，重点收录美国版权过期的旧作品。
2. Standard Ebooks：基于公版文本整理的高质量开放电子书。
3. Wikisource / Chinese Text Project：古籍、公版文本与文献入口。
4. Open Library / WorldCat / 出版社官网：现代版权书的合法检索、借阅、购买或馆藏入口。

## 注意

- 同一作品的“原文公版”不代表所有译本、注释本、排版本都可自由上传。
- 中文现代译本、点校本、注释本通常仍受版权保护。
- 对现代书，本仓库只保存书目信息和合法入口，不保存全文。
