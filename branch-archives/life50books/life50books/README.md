# life50books

这是一个 50 本重要书籍的合法资源索引分支。

## 内容

- `data/life50_books.csv`：结构化 CSV 书目与资源入口。
- `data/life50_books.json`：结构化 JSON 书目与资源入口。
- `books/`：每本书一页，包含为什么重要、合法资源入口和版权提醒。
- `sources/legal_sources.md`：使用的合法资源站点说明。
- `sources/copyright_policy.md`：版权处理规则。

## 版权原则

本仓库不上传现代受版权保护书籍的全文、扫描 PDF、TXT 或转录内容。公版作品只放合法公开入口；现代作品只放官方、出版社、图书馆或目录检索入口。
